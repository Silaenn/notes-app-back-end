# 📟 CyberNote Y2K

**CyberNote Y2K** adalah aplikasi manajemen catatan (*note-taking*) dengan estetika retro-futuristik tahun 2000-an (Y2K). Aplikasi ini mensimulasikan pengalaman menggunakan sistem operasi klasik lengkap dengan sistem manajemen jendela, taskbar, dan pemutar musik terintegrasi.

## ✨ Fitur Utama

- **🖥️ Multi-Window OS Interface**: Pengalaman desktop simulasi di dalam browser.
- **📥 Minimize to Taskbar**: Jendela bisa di-*minimize* ke Taskbar dan dikembalikan (*restore*) persis seperti OS sungguhan.
- **📝 Advanced Note Editor**: Buat, edit, dan hapus catatan dengan metadata tags.
- **🔍 Smart Note Explorer**: Cari catatan Anda dengan fitur pencarian instan.
- **🎶 Winamp Player**: Nikmati musik latar bertema *ambient-cyber* dengan pemutar musik ala Winamp klasik.
- **📱 Ultra-Responsive UX**: Tampilan yang dioptimalkan untuk desktop maupun mobile (Full-screen mobile mode).
- **🕹️ Aesthetic Y2K**: Scanlines, font pixel, dan skema warna *teal* yang ikonik.

## 🚀 Teknologi yang Digunakan

- **Frontend**: React.js, Tailwind CSS, Framer Motion (untuk animasi jendela).
- **Backend**: Node.js, Hapi.js.
- **Icons**: Lucide React.
- **Fonts**: VT323 (Terminal), Press Start 2P (Pixel).

## 🛠️ Cara Instalasi

1. Clone repositori ini:
   ```bash
   git clone https://github.com/username/cybernote-y2k.git
   ```
2. Masuk ke direktori proyek:
   ```bash
   cd cybernote-y2k
   ```
3. Instal dependensi:
   ```bash
   npm install
   ```
4. Jalankan aplikasi (Development mode):
   ```bash
   npm run dev
   ```

## 🎮 Cara Menggunakan

- **Buka Jendela**: Klik ikon di desktop (My Notes, New Note, atau Winamp).
- **Manajemen Jendela**: Gunakan tombol di pojok kanan atas jendela untuk meminimalkan, memperbesar, atau menutup.
- **Taskbar**: Klik tombol di taskbar untuk berpindah antar jendela yang sedang terbuka atau mengembalikan jendela yang diminimalkan.
- **Mobile**: Pada layar HP, jendela akan otomatis menjadi *full-screen* untuk kenyamanan navigasi.
