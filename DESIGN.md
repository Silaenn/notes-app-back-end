# CyberNote OS — Design Rationale

## Overview
Estetika **Retro Desktop OS** terinspirasi dari Windows 95/98 namun dengan sentuhan modern 2026. Menggunakan elemen skeuomorphic, window management, dan palette warna pastel yang cerah.

## Visual Dials
- **DESIGN_VARIANCE**: 9/10 (Desktop-within-a-browser).
- **MOTION_INTENSITY**: 4/10 (Snappy transitions, draggable windows).
- **VISUAL_DENSITY**: 7/10 (Banyak jendela kecil/floating elements).

## Key Elements
- **Taskbar Top**: Menu navigasi dan sistem clock berada di bagian atas.
- **Classic Windows**: Jendela dengan title bar abu-abu, tombol kontrol (min, max, close), dan border tebal (inset/outset).
- **Desktop Icons**: Ikon di area desktop untuk akses cepat.
- **Pastel Gradients**: Background desktop menggunakan gradasi pink-ungu lembut.
